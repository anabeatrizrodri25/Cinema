# 💕 Convite Especial para o Cinema

Mini-site romântico e interativo criado como um convite para ir ao cinema com Byanca Eduarda.

## 🎬 Características

- **Design Romântico**: Fundo escuro estilo cinema com gradientes suaves em rosa e dourado
- **Mobile-First**: Totalmente responsivo, otimizado para dispositivos móveis
- **4 Telas Interativas**: Transições suaves com fade e deslizamento
- **Animações Especiais**: 
  - Botão "Não" que foge do mouse (brincadeira clássica)
  - Confetes e corações quando aceita o convite
  - Micro-animações nos botões
- **Tipografia**: 
  - Poppins para textos principais (moderna e limpa)
  - Caveat para o recadinho no post-it (estilo manuscrito)

## 🌐 Como Acessar

### Opção 1: Versão React (Online)
Acesse diretamente: **https://movie-night-ask.preview.emergentagent.com**

### Opção 2: Arquivo HTML Standalone
O arquivo `convite-cinema.html` na raiz do projeto (`/app/convite-cinema.html`) pode ser:
- Aberto diretamente no navegador
- Hospedado em qualquer servidor web
- Compartilhado facilmente

Para abrir localmente, basta fazer download do arquivo e abrir no navegador.

## 📱 Fluxo das Telas

1. **Tela 1**: Convite inicial com foto do cinema Cineflix
   - "Byanca Eduarda... Vamos cineminha comigo?"
   - Botões: Sim / Não (que foge!)

2. **Tela 2**: Referência aos Jogos Vorazes
   - "Não é jogos vorazes, mas eu prometo que vai ser um bom filme..."

3. **Tela 3**: Promessa do filme Devorador de Estrelas
   - "Mas eu prometo que vai ser um bom filme, e se for ruim a gente fica conversando..."

4. **Tela 4**: Declaração final
   - "Só eu tenho saudadessss"
   - Imagem fofa do Flork com rosa
   - Botões: "Eu também tenho!" (confetes + corações) / "Não tenho" (foge!)
   - Post-it com recadinho carinhoso

## 🎨 Paleta de Cores

- **Background**: Gradiente escuro (#1a0e1e → #2d1b2e)
- **Acentos**: Rosa suave (#ffb6c1) + Dourado (#ffd700)
- **Textos**: Tons claros (#f5f5f5, #e8d5e8)
- **Botões**: 
  - Sim: Gradiente rosa
  - Avançar: Gradiente dourado
  - Não: Transparente com borda

## 💻 Tecnologias

- HTML5
- CSS3 (Animações, Gradientes, Backdrop Filter)
- JavaScript Vanilla
- React (versão online)
- Google Fonts (Poppins, Caveat)

## ✨ Recursos Especiais

- Textura de ruído sutil no fundo
- Glassmorphism nos cards
- Animações de confetes e corações
